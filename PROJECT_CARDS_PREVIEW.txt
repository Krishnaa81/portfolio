                        DESIGN FEATURES
═══════════════════════════════════════════════════════════════════════

🎨 VISUAL EFFECTS:
  ✨ Gradient Backgrounds
     - Blue → Cyan (Library)
     - Purple → Pink (SAM)
     - Green → Teal (Parking)

  ✨ Animated Grid Pattern
     - Moving diagonal lines
     - Subtle animation
     - 20s loop duration

  ✨ Large Emojis
     - 📚 Books (Library System)
     - 🍽️ Plate (Food Sharing)
     - 🚗 Car (Parking)
     - 6rem size (very large!)

  ✨ Hover Animations
     - Card scales to 1.05x
     - Border glows purple
     - Icon scales to 1.10x
     - Smooth transitions (300ms)

🎯 LAYOUT:
  • Height: 48rem (taller cards)
  • Border radius: Rounded XL
  • Border: Dark with purple glow on hover
  • Background: Dark gradient card
  • Spacing: 8 unit gap between cards

📱 RESPONSIVE:
  • Mobile: 1 column (stacked)
  • Tablet: 2 columns
  • Desktop: 3 columns (side by side)


═══════════════════════════════════════════════════════════════════════
                     COMPONENT STRUCTURE
═══════════════════════════════════════════════════════════════════════

Each project card has:

1. HEADER IMAGE (48rem height)
   ├─ Gradient background (unique color per project)
   ├─ Animated grid overlay (moving pattern)
   ├─ Large emoji icon (centered, animated)
   └─ Project title (white text, centered)

2. CONTENT SECTION
   ├─ Project title (large, bold)
   ├─ Description (gray text, 4 lines)
   ├─ Tech stack tags (rounded pills)
   └─ Action buttons (GitHub + Demo)

3. INTERACTIVE ELEMENTS
   ├─ Hover effect (scale + glow)
   ├─ Gradient brightness change
   ├─ Icon zoom effect
   └─ Button animations
