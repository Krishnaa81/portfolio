## 💡 Pro Tips

1. **Use Find & Replace**
   - Replace `"Your Name"` with your actual name
   - Replace `your.email@example.com` with your email
   - See `LINKS_TO_UPDATE.md` for all replacements

2. **Test as You Go**
   - Keep `npm run dev` running
   - Save files to see instant updates
   - Check mobile view (F12 → Toggle device toolbar)

3. **One Section at a Time**
   - Update Hero first (most visible)
   - Then About, then Projects
   - Don't overwhelm yourself!

4. **Keep It Simple**
   - Don't overcomplicate your descriptions
   - Short and clear is better than long and complex
   - Let your projects speak for themselves

## 🚨 Common Issues & Solutions

### Issue: Port already in use
```bash
# Use a different port
npm run dev -- --port 3000
```

### Issue: Changes not showing
- Hard refresh: `Ctrl + F5` (Windows) or `Cmd + Shift + R` (Mac)
- Clear cache and reload
- Check if file is saved

### Issue: Module not found
```bash
# Reinstall dependencies
npm install
```

### Issue: Can't find resume.pdf
- Make sure it's in `public/` folder
- Name it exactly: `resume.pdf` (lowercase)
