OPTION A: Drag & Drop (No Command Line!)
─────────────────────────────────────────────────────────────────────

1. Go to: https://vercel.com

2. Click "Sign Up" 
   → Sign in with GitHub (easiest)

3. Click "Add New" → "Project"

4. Click "Browse" and select your "portfolio/dist" folder
   OR drag the "dist" folder to the upload area

5. Click "Deploy"

6. DONE! You'll get a URL like:
   https://portfolio-xxxx.vercel.app

7. Copy and share your URL anywhere! 🎉


OPTION B: Using Command Line (Fastest!)
─────────────────────────────────────────────────────────────────────

Open Terminal and run:

   npm install -g vercel
   cd portfolio
   vercel

Follow the prompts:
  - Login? Yes (opens browser)
  - Set up and deploy? Yes
  - Which scope? Select your account
  - Link to existing project? No
  - Project name? krishna-portfolio
  - Directory? ./ (press Enter)
  - Override settings? No

Wait 1-2 minutes...

DONE! You'll get:
   https://krishna-portfolio.vercel.app
